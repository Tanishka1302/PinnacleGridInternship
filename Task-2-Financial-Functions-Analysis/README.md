# Financial Functions Analysis Using Excel & Power BI

## Objective

Analyze financial data using Excel and Power BI by calculating financial functions and creating an interactive dashboard.

## Tools Used

- Microsoft Excel
- Microsoft Power BI

## Financial Calculations

- Net Present Value (NPV)
- Return on Investment (ROI)
- Payback Period
- Total Investment
- Total Returns
- Net Profit

## Dashboard Features

- KPI Cards
- Year-wise Cash Flow Analysis
- Profitability Analysis
- Investment Summary

## Key Insights

- Positive NPV indicates a profitable investment.
- ROI is 117.38%.
- Total Returns exceeded Total Investment.
- Net Profit is ₹5869.
- Payback Period is approximately 2.30 years.
