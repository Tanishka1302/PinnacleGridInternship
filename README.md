# Task 1 - Regional Sales Analysis

## Project Overview
This project analyzes regional sales performance using Microsoft Power BI. The dashboard provides insights into Sales, Profit, and Units Sold across different regions.

## Objectives
- Analyze regional sales data
- Compare sales and profit across regions
- Identify high and low performing regions
- Build an interactive Power BI dashboard

## Dashboard Features
- KPI Cards
  - Total Sales
  - Total Profit
  - Total Units Sold

- Sales by Region
- Profit by Region
- Units Sold by Region
- Sales vs Profit by Region
- Region Slicer

## Tools Used
- Microsoft Power BI
- Microsoft Excel

## Dataset
Regional Sales Dataset

## Business Insights
1. West region generated the highest sales.
2. West region earned the highest profit.
3. East region ranked second in sales performance.
4. North region recorded the lowest sales and profit.
5. Higher sales generally resulted in higher profit.
