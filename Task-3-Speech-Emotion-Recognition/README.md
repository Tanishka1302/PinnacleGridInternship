# Speech Emotion Recognition System

## Overview
This project is a Speech Emotion Recognition System built using Python and Machine Learning. It detects human emotions from speech audio files using MFCC features and a Random Forest Classifier.

## Features
- Detects emotions from speech recordings
- Uses MFCC feature extraction
- Trained on the RAVDESS dataset
- Predicts emotions from WAV audio files

## Technologies Used
- Python
- Librosa
- NumPy
- Scikit-learn
- Joblib

## Dataset
RAVDESS Emotional Speech Audio Dataset

## Model
Random Forest Classifier

## Accuracy
61.11%
