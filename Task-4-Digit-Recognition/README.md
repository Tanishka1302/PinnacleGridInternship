# Handwritten Digit Recognition System Using Python

## Project Overview

This project is a Machine Learning application that recognizes handwritten digits (0–9) using the MNIST dataset. A Neural Network model is trained using TensorFlow and Keras to classify handwritten digit images with high accuracy.

## Features

- Recognizes handwritten digits (0–9)
- Uses the MNIST dataset
- Neural Network implemented using TensorFlow/Keras
- Achieves around 97–98% accuracy
- Displays predictions with the actual digit
- Saves training accuracy graph

## Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## Dataset

MNIST Handwritten Digit Dataset

- Training Images: 60,000
- Testing Images: 10,000
